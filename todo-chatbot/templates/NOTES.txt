Your Todo Chatbot application has been deployed with:

- Frontend: {{ .Values.frontend.replicaCount }} replica(s) on port {{ .Values.frontend.service.port }}
- Backend: {{ .Values.backend.replicaCount }} replica(s) on port {{ .Values.backend.service.port }}

To access your Todo Chatbot application:

1. Get the frontend service external IP/URL:
   NOTE: It may take a few minutes for the LoadBalancer IP to be available.
   You can watch the status by running:
   
   kubectl get svc todo-frontend-service --namespace {{ .Release.Namespace }} -w

2. Once the external IP is available, visit:
   http://<EXTERNAL-IP>:{{ .Values.frontend.service.port }}

3. The backend API is available internally at:
   http://todo-backend-service:{{ .Values.backend.service.port }}

To verify the deployment:
  kubectl get pods --namespace {{ .Release.Namespace }}

To upgrade the deployment:
  helm upgrade {{ .Release.Name }} . --namespace {{ .Release.Namespace }}

To uninstall the deployment:
  helm delete {{ .Release.Name }} --namespace {{ .Release.Namespace }}
